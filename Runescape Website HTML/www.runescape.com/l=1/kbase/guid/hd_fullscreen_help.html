HD startet nicht




Schritte zur Problembehebung

Wenn High Detail auf Ihrem Computer nicht richtig funktioniert (sich nicht starten lässt, abstürzt, langsam läuft etc.), versuchen Sie es mit einer der folgenden Lösungen:

  • Überprüfen Sie, ob Ihr Computer die Mindestanforderungen für High Detail erfüllt.
    Für RuneScape High Detail sind mindestens 256MB RAM, ein 1.5 GHz Prozessor und eine Grafikkarte wie Nvidia GeForce 3 (oder Äquivalent) erforderlich. Klicken Sie hier, um zu überprüfen, ob Ihr Computer kompatibel ist.

  • Starten Sie Ihren Rechner neu.
    Wenn der High-Detail-Modus normalerweise auf Ihrem Rechner funktioniert, sollten Sie Ihren Computer neu starten. Das beseitigt das Problem oft!

  • Java aktualisieren.
    Klicken Sie hier, um die neuste Java-Version herunterzuladen und zu installieren.

  • Laden Sie die aktuellen Windows-Updates herunter.
    Klicken Sie hier, um die Windows-Update-Seite zu besuchen, wo Sie die neusten Updates für Ihr Betriebssystem herunterladen können.

  • Installieren Sie den aktuellen Grafikkartentreiber.
    Besuchen Sie die Webseite des Herstellers Ihrer Grafikkarte oder Ihres Computers, um aktuelle Treiber für die Grafikkarte zu finden, und installieren Sie diese. Klicken Sie hier, um herauszufinden, welcher Grafikkartentreiber auf Ihrem Computer installiert ist. Es gibt drei große Hersteller von Grafikchipsets, deren Websites Sie hier finden:

    http://www.nvidia.com/Download/index.aspx?lang=de - für Nvidia-Karten
    http://ati.amd.com/support/driver-de.html - für ATI-basierte Grafikkarten
    http://downloadcenter.intel.com/default.aspx?lang=deu# - für Intel-Grafikkarten


  • Ändern Sie Ihre Java-Einstellungen.
    Auf unserer Website unter 'Jetzt spielen' finden Sie den Menüpunkt 'Java-Optionen'. Ändern Sie diese auf 'Sun Java erzwingen', speichern Sie diese Einstellung und laden Sie das Spiel wie gewohnt.

  • Wenn Sie Windows 2000 oder XP benutzen, überprüfen Sie, ob die Hardwarebeschleunigung aktiviert ist.
    1. Rechtsklicken Sie eine leere Stelle auf dem Desktop.
    2. Wählen Sie 'Eigenschaften' im Kontextmenü.
    3. Öffnen Sie die Registerkarte 'Einstellungen'.
    4. Klicken Sie auf die Schaltfläche 'Erweitert'.
    5. Öffnen Sie die Registerkarte 'Problembehandlung'.
    6. Hier sollte es einen Schieberegler mit der Beschriftung 'Hardwarebeschleunigung' geben. Stellen Sie sicher, dass dieser auf 'Voll' steht und klicken Sie 'OK'.

  • Stellen Sie sicher, dass Ihre 3D-Grafikkarte verfügbar ist.
    Sie können den High-Detail-Modus nicht benutzen, wenn bereits ein Browserfenster offen ist, das die 3D-Grafikkarte beansprucht - z.B. wenn Sie versuchen, das Spiel zwei Mal gleichzeitig zu laden.

  • Benutzen Sie die aktuelle Version von Firefox.
    Laden Sie hier die aktuelle Firefox-Version herunter und benutzen Sie diese zum Spielen.

  • Versuchen Sie, den RuneScape-Windows-Client zu verwenden.
    Versuchen Sie, den RuneScape-Windows-Client herunterzuladen und damit das Spiel ohne Browser zu starten. Der Windows-Client benötigt mindestens Internet Explorer 6.

  • Schalten Sie selbstgemachte / animierte Mauszeiger ab.
    Einige veränderte oder animierte Mauszeiger können beim Spielen Probleme verursachen. Wenn Sie im HD-Modus Probleme mit verschwindenden Mauszeigern haben, sollten Sie wieder die normalen Windows-Mauszeiger einstellen. Gehen Sie zu 'Systemsteuerung', 'Maus', 'Zeiger' und wählen Sie das Schema 'Kein' aus.

  • Überprüfen Sie, ob Sie 32-Bit-Farbqualität verwenden.
    Sollten Ihnen die RuneScape-Farben etwas merkwürdig vorkommen, überprüfen Sie, ob Ihr Computer im 32-Bit-Modus läuft.

    Unter Windows 2000 oder XP:
    1. Rechtsklicken Sie eine leere Stelle auf dem Desktop.
    2. Wählen Sie 'Eigenschaften' im Kontextmenü.
    3. Öffnen Sie die Registerkarte 'Einstellungen'.
    4. Stellen Sie 'Höchste (32 Bit)' ein.

    Unter Windows Vista:
    1. Rechtsklicken Sie eine leere Stelle auf dem Desktop.
    2. Wählen Sie 'Personalisieren' im Kontextmenü.
    3. Klicken Sie auf 'Anzeigeeinstellungen'.
    4. Wählen Sie unter 'Farben' die Option 'Höchste (32 Bit)'.

    Unter Mac OS:
    1. Klicken Sie auf das Apple-Symbol in der oberen linken Ecke Ihres Bildschirms.
    2. Wählen Sie 'Systemeinstellungen'.
    3. Wählen Sie 'Monitore' aus.
    4. Unter 'Farbe', wählen Sie Millionen aus.

  • Stellen Sie Talkback in Firefox ab.
    Wenn Sie RuneScape High Detail in Firefox spielen und Leistungsprobleme haben, sollten Sie versuchen, Talkback zu deaktivieren. Talkback ist eine Erweiterung für Firefox, die standardmäßig installiert ist und Informationen über Abstürze von Firefox an Mozilla sendet. Es gibt mit Talkback ein bekanntes Problem, wegen dem Java-Programme, die OpenGL benutzen, massiv verlangsamt werden können. Um Talkback zu deaktivieren, wählen Sie im Menü 'Extras' den Punkt 'Add-Ons', wählen 'Talkback' in der erscheinenden Liste aus und klicken auf 'Deaktivieren'.

  • Spielen Sie RuneScape auf der normalen Detailstufe oder im Sicherheitsmodus.
    Falls Sie weiterhin Probleme haben, RuneScape High Detail zu starten, können Sie RuneScape immer noch auf der normalen Detailstufe oder im Sicherheitsmodus spielen.
