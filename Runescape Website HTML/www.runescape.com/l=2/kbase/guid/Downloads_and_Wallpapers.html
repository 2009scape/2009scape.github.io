Téléchargements et Fonds d'écran


Téléchargements
Les fichiers supplémentaires suivants peuvent être téléchargés.

Remarque : vous n'avez besoin de télécharger aucun de ces fichiers pour jouer à RuneScape. Vous pouvez jouer directement depuis votre navigateur Internet en cliquant sur "Jouer à RuneScape" sur la page d'accueil !

Les fichiers suivants sont des bonus offerts en plus du jeu aux utilisateurs qui le souhaitent.

Il s'agit des seuls téléchargements officiels de RuneScape. Les téléchargements non officiels provenant d'autres sites peuvent contenir des virus, des enregistreurs de frappe, etc. Ainsi, si vous voulez jouer hors de votre navigateur, nous vous recommandons de n'utiliser le client qu'à partir d'ici pour garantir la sécurité de votre compte.
Client Windows
  • Lancez RuneScape directement à partir de votre bureau, sans passer par votre navigateur ;
  • Inutile d'installer Java - idéal si vous ne pouvez pas installer Java ou en cas de problèmes avec Java ;
  • Aucun privilège administrateur requis sur votre ordinateur ;
  • Application « open source». Pour une copie, cliquez ici.
Cliquez ici pour télécharger. Si vous rencontrez des difficultés, consultez notre guide de dépannage.

Cartes de duel RuneScape

10 cartes de duel RuneScape à télécharger et à imprimer pour jouer contre vos amis !

Vous êtes-vous déjà demandé laquelle des plus grosses créatures de RuneScape l'emporterait sur les autres ? Pour le savoir, téléchargez et imprimez les 10 cartes de duel (lien ci-dessous) et faites s'affronter les puissantes créatures de RuneScape ! Le but du jeu est de vaincre votre adversaire en remportant ses cartes grâce aux points d'Attaque, de Magie, de Combat à distance, d'Agilité ou de RuneScape. Le règlement complet est consultable sur le forum : 1-2-12-3432
