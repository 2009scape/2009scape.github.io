Aide

Vous pouvez associer une adresse électronique à votre compte Jagex lorsque vous créez votre compte Jagex. Cette page explique comment l'enregistrer ou la modifier.

Adresse électronique

Ce service courriel nous permettra de vous tenir informé des futures mises à jour et de vous offrir un meilleur service d'assistance. Pour enregistrer votre adresse électronique, cliquez ici.


Pour modifier ou supprimer votre adresse électronique, il suffit de cliquer ici. Une fois que vous avez enregistré votre nouvelle adresse, un code de confirmation sera envoyé à votre ancienne adresse afin de vérifier votre identité. Un code de vérification sera également envoyé à votre nouvelle adresse et vous sera demandé sur notre site.

Paramètres courriel

À la page des Paramètres courriel, vous pouvez modifier vos préférences et choisir d'annuler un service ou mise à jour particulière. Vous devrez confirmer en envoyant le code de confirmation.



Code de confirmation

Lorsque vous enregistrez votre adresse électronique ou que vous changez vos préférences courriel, il vous sera envoyé un code de confirmation que vous devrez ensuite saisir sur notre site en cliquant sur le lien inclus dans le courriel ou en visitant cette page. Ce code ne vous sera demandé qu'une seule fois.
