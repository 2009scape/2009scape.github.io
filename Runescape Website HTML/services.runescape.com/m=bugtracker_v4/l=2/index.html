Signaler un bug

Catégorie Bugs courants Faux bugs Connexion Fin du rapport Vérification Envoyer
Sélectionnez une catégorie
Ce formulaire permet de nous signaler un bug. Nous faisons tout pour limiter le nombre de défauts ou de problèmes dans le jeu, mais certains nous échappent ! Vous pouvez nous aider à les éliminer. Il vous suffit de choisir la catégorie correspondant le mieux au problème que vous avez rencontré. Si vous avez besoin d'aide pour une quête ou que vous souhaitez faire part de votre opinion sur nos services, utilisez les formulaires conçus dans ce but à la page principale.

Les problèmes décrits ci-après sont des bugs :
  • Être bloqué dans une quête impossible à terminer
  • Être bloqué dans une partie du décor ou défaut graphique dans le décor
  • Animations défectueuses
  • Fautes d'ortographe ou de grammaire (catégorie Texte)
Ne signalez pas un bug lorsque vous êtes simplement coincé dans une quête et ne savez pas que faire pour continuer. Dans ce cas, consultez notre QuestHelp (en anglais). Les bugs concernant RuneScape Classic signalés ne seront pas examinés.
