Enviar um relatório de erro

Categoria Erros comuns Casos equivocados Login Fazer relatório Revisar relatório Enviar
Selecione uma categoria:
Boas-vindas à página de relatórios de erros. Trabalhamos muito para garantir que o jogo esteja livre de erros o quanto possível e dependemos de relatórios de usuários para mantê-lo assim. Para continuar, selecione a categoria que acha que melhor descreve o problema que encontrou. Se você somente quer ajuda em uma missão, ou gostaria de comentar sobre o serviço que oferecemos, use os formulários apropriados cujos links encontram-se na página principal.

O tipo de coisa que consideramos como erros são:
  • Pontos em uma missão onde você não pode prosseguir devido a uma missão que se tornou impossível de completar
  • Áreas do mapa onde fique preso ou que não pareçam corretas
  • Erros gráficos, tais como animações que não funcionam adequadamente
  • Erros de ortografia e gramática (submeta-os sob a categoria "Redação")
Não envie um relatório através deste sistema se você está preso em uma missão e não sabe o que fazer a seguir. Se este for o caso, use nosso sistema de Questhelp (em inglês) para dicas. NÃO envie erros do RuneScape Clássico. Nossa equipe de testes SOMENTE revisará erros do RuneScape.
