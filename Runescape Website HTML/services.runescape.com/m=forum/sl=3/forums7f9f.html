¡Necesito un Jmod!
Buenas a todos!

Este hilo nos servirá para comunicarnos un poco a la hora de moderar los foros Hispano Hablantes.

Será básicamente un sitio donde podais “reportar” a la gente que postee rompiendo las reglas del foro como son:
- Insultos graves, conversaciones de sexo, racismo, etc.
- Estafas relacionadas con contraseñas o objetos.
- Compra/Venda de cuentas.
- Nombrando URL sospechosas.
- Dando datos personales como son edad o donde vivís.
- Spam

Aquí os dejo una plantilla para que podáis copiar y pegar, llenáis la información y listo!

Título del hilo:

_____________________________

Cuando lo haya leído y tomado acción, esconderé el post.

Pero aparte de eso, este hilo también servirá como puente para comunicaros conmigo. Como podreis comprender, no puedo estar 24h en Jagex, también necesito mis horas de descanso!

Cualquier cosa que necesiteis decirme, no dudeis en postear. Vuestro post también será escondido automáticamente para mas privacidad.

Salu2!
