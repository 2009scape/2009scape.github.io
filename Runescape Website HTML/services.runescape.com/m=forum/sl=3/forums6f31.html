Creación de Eventos

Cuando hayas posteado tu evento detalladamente, con toda la información de que se tratará, puedes postearme la información básica para que lo añada aquí. De esta forma, si alguien quiere ver rápidamente que eventos están disponibles, podrá verlo en este hilo (arriba en los Eventos Pendientes).
Unos consejos muy básicos para quien no haya organizado nunca un evento:

Guia Para Organizar Eventos
  • ¡Simplicidad! Si lo haces demasiado complicado, la gente pierde interés.
  • Centrarse en algo! Da algo para que la gente pueda participar. Que no solo sea una escusa para chatear.
  • Lugar apartado! No hace falta que diga que hacer un evento en medio de la GE no tendrá mucho éxito.
  • Evita el spam! En el caso de que quieras organizar algo en un sitio con mucha gente como la GE, crea un ClanChat.
  • Accesibilidad! Como mas difícil sea llegar o participar, menos gente irá. Está bien organizar eventos para gente de nivel alto, pero si se necesita algún requerimiento (quest, nivel, equipo, etc) déjenlo claro.

Plantilla Para Crear Evento

EVENTO: - nombre del evento.
FECHA: - dia, mes y año del evento.
HORA: - hora en UK/Jagex/Forum del evento.
SERVER: - en que server empieza el evento.
QUEDADA: - donde planeas quedar para empezar el evento.
LINK: - el link del thread de tu evento.
CC: - Clan Chat donde se reunirán para hacer el evento



Mucha suerte y espero ver eventos pronto!
