Hjälp-forumet

Kontoproblem

För att hålla vår spelarcommunity säker och hålla fusk ute ur spelet har vi ett system som registerar kontoaktiviteter relaterade till regelbrott för. Vanligtvis ges förseelser till konton när en spelare blir rapporterad i spelet och en medlem av Jagexs personal ser över bevisningen och beslutar att en regel har brutits.

Vi tillåter vanligtvis spelare att överklaga förseelsen på deras konto, ifall vi har begått ett misstag eller det finns en förklaring för regelbrottet som vi borde överväga.

För att hantera förseelser på ditt konto, inklusive att överklaga dem där detta alternativ finns, var vänlig och använd nedanstående länk.

Överklaga en förseelse

Om du kan ditt nuvarande lösenord till ditt konto, och du vill ändra det, vänligen använd nedanstående länk.

Ändra ditt lösenord

Om du har glömt ditt lösenord, eller om du tror någon annan fått kännedom om det, kan du ställa in ett helt nytt lösenord med nedanstående länk.

Glömt eller stulet lösenord?

Om vi behöver skicka någon viktigt information kommer vi skicka den till din kontoinbox. Använd länken nedan för att ta dig till Meddelande Centret:

Konto – Läs Meddelanden

Om ditt konto har låsts kan du starta upplåsningsprocessen med nedanstående länk.

Återfå ett låst konto

Om ditt konto har blivit permanent bannlyst eller fått sin chatfunktion permanent borttagen erbjuder vi möjligheten att få konton undersökt av ett specialistteam. De kontrollerar alla förseelser och ser om de kan göra något för att ge tillbaka kontot eller dess chatfunktioner.

För att komma åt denna funktion behöver du posta namnet på kontot på “Account Review Thread”-tråden – vilken är en olåst klistrad tråd i INAM-forumet.

Du kan också nå denna tråd genom att använda quick find code 103-104-5-59725648 i “Jump to thread”-boxen högst upp på forumen.

Då och då låser vi “Account Review Thread”-tråden medan vi jobbar ikapp med att gå igenom alla konton. Om tråden är låst när du besöker den, återkom igen om ett par dagar.


Betalningsproblem

Om du vill starta en teckna medlemskap för ditt konto, var vänlig och använd denna länk för att komma igång.

Om du redan är medlem, eller vill ta upp ett problem relaterat till betalningen för ditt konto, var vänlig och använd vårt betalningshjälpsystem.

Vi vet att problem relaterade till ditt kontos medlemskap är väldigt viktiga. Av denna anledningen tillhandahåller vi mycket information på vår webbsida, och vi har också en tråd med svaren på de vanligaste frågorna om detta. Denna tråd hittar du med quick find code (QFC) 222-223-12-60839254.

Om du har försökt med denna tråd, och använt vårt betalningshjälpsystem som nämns ovan, och du fortfarande inte kan hitta lösningen på din fråga, då kan du även kontakta vårt betalningsteam direkt via email. Vi uppskattar om du endast använder emailalternativet som en “sista utväg”, och i all fall, endast för betalningsrelaterade frågor.

För att kontakta betalningsteamet, skicka ett email till billingsupport@jagex.com eller klicka på denna direktlänk. Det går utmärkt att skicka email även på svenska!


TEKNISKT – BUGGAR

Om du hittar en bugg i våra spel rekommenderar vi att du inte utnyttjar den för egen vinning eller berättar för andra spelare om den.

Istället skulle vi verkligen uppskatta om du skickar in en buggrapport till oss. När du gör detta går rapporten direkt till våra specialistteam som påbörjar arbetet med att ställa saker tillrätta så snabbt som möjligt.

För att informera oss om en möjlig bugg, vänligen använd Submit a bug report-systemet på vår webbsida.

Om du har ett tekniskt problem som inte är en bugg kan du kontakta oss med detaljerna. Om vi kan så kommer vi hjälpa dig att lösa problemet!
