Har du någon gång haft problem med att köra RuneScape? Det kanske har kommit upp ett error meddelande som inte vill ge med sig, hur mycket du än försöker. Det finns några knep som eventuellt hjälper mot det här.

Det första man skall försöka med är att starta om sin webbläsare och, eller sin dator.

Din Java version kanske är gammal.

Dina drivrutiner till ditt grafikkort är gamla.

Din dator kanske är för långsam?

Vad skall du då göra?

Eftersom RuneScape är gjort för Windows och Internet Explorer är den här "guiden" skriven för endast Windows. Jag använder en engelskspråkig Windows 7, och där med är alla platser hur de förekommer i Windows 7. Platserna i Windows XP och Windows Vista kommer snart.

=

Min Java version är gammal. Hur skall jag uppdatera den?

Eftersom du har en gammal Java version, finns det en uppdatering på Sun Microsystems Java sidor. Det lönar sig först att avinstallera alla gamla Java versioner som du har på din dator och sedan installera den nyaste versionen. Detta kan du göra genom att gå till Control Panel -> Programs and Features. Här söker du upp "Java (TM) 6 Update ##" och avinstallerar. Efter det här kan du gå till Javas webbsida och välja "Free Java Download". Vill du kolla vilken Java version du har är det bara att klicka på "Do I have Java?"

=

Vad skall jag göra om mina drivrutiner till mitt grafikkort är gamla?

De två vanligaste märkena på grafikkort är NVidia och ATi. Du måste helt enkelt gå in på någon av deras sidor beroende på vilket grafikkort du har.


Hur får jag reda på vilket typ av grafikkort jag har?

Öppna Kör (Run) och skriv in "Dxdiag" i rutan och tryck på Enter. Under tabben "Display" hittar du vilket grafikkort du har.

Min dator är långsam. Vad skall jag göra?

Är din dator långsam brukar det oftast bero på att utrymmet på din hårdskiva håller på att ta slut eller att du har för lite RAM (Random Access Memory) - minne. Du kanske också har haft en massa filer på din dator och sedan raderat dem.
Det finns några knep för att frigöra minne på dator. Det första är att radera alla temporära internet filer. Dessa tar upp en massa utrymme på hårdskivan utan att du vet om det. Nedan följer en steg för steg anvisning hur man tar bort dessa filer från datorn.

Internet Explorer
1. Öppna webbläsaren
2. Gå till Tools (verktyg) och därifrån till Internet Options
3. Här hittar du, under General, något som heter "Browsing history". Tryck på Delete...
4. Starta om webbläsaren


Mozilla Firefox
1. Öppna
2. Gå till Tools -> Clear Recent History
3. Välj vad du vill ta bort
4. Starta om webbläsaren

Förutom att du kan ta bort de Temporära internet filerna som ovan, finns det också några program som går att använda.

Då du har tagit bort filer från din dator har det lämnat så kallade hål i din hårdskiva. Dessa hål bidrar också till att din dator fungerar långsammare än vanligt. Detta går att korrigera genom att defragmentera datorn. Du påbörjar en defragmentering genom att gå till Start > All Programs > Accessories > System Tools > Disk Defragmenter. Var noga med att du inte väljer "Format" eller något liknande. Detta alternativ kommer att radera allt som finns på din dator, även operativ systemet och du måste du använda en Recovery CD för att köra in operativ systemet igen på din dator och därefter följer installation av alla program som funnits på din dator. Den senaste RuneScape uppdateringen fodrar att man har åtminstone DirectX version 9.0 om man vill spela RuneScape med DirectX grafik. Hur får jag reda på vilken version av DirectX jag har?

Start > Kör och skriv in "dxdiag"

Detta ger dig en ruta där det under "System" tabben finns en text med "DirectX version:"

Är din version gammal kan du försöka med att ladda ner den nyaste versionen.

=

Vad skall jag göra om inte HD/lobbyn fungerar? Vid ett error meddelande?

Du kan pröva med att ta bort en mapp som heter .jagex_cache_32 eller .jagex_cache_34

Du borde hitta den här mappen under c:\users\namn eller under c:\windows . I samma mapp kan det hända att du hittar någon eller alla av följande filer:

jagex_preferences3.dat
jagex_runescape_preferences.dat
jagex_runescape_preferences2.dat

Radera de här också.

Öppnar du .jagex_cache_32 - mappen hittar du en till mapp som borde heta "runescape". Öppnar du den hittar du troligtvis två filer vid namn "jaggl.dll" och "jagmisc.dll". Dessa är inget att bry sig om eftersom de kommer att laddas till din dator på nytt då du laddar om RuneScape. Har du spelat FunOrb finns där en mapp som heter "funorb".

I samma mapp där .jagex_cache_32 mappen var, gör du en ny mapp, genom att högerklicka och välja new -> folder, vid namn rs_cache .


Starta om din webbläsare och pröva att köra RuneScape.

Fungerar det - bra så, fungerar det inte måste vi hitta på något annat roligt
