Effigy Assistance
Please use this sticky to request help with an Effigy. When doing so, it would be most helpful if you used the following format:

Skill and Lvl Required:
Date and Time:
Preferred Location and World:
Contact via PM, CC or response post:
Additional Requirements/Information:


Please edit your post to read 'Assisted' once you have been helped, to ensure the thread runs smoothly.

Good luck and happy gaming!

NOTE: PLEASE DO NOT POST OPEN OFFERS ON THIS THREAD, AS THIS CREATES LOTS OF ADDITIONAL, SOMETIMES UNNEEDED POSTS. IF YOU WISH TO HELP SOMEONE, READ THROUGH THE LAST FEW PAGES AND I'M SURE YOU CAN FIND SOMEONE WHO IS IN NEED OF YOUR SERVICES.

Please note that this sticky is a test to gauge its need, however if it is deemed useful it will become a permeanent fixture.
