Thanks for taking the time to compile this thread - it is extremely beneficial to the community

Welcome to the Guide Directory. The purpose of this thread is to have an organized list of the guides that people wrote and a quick and easy way to find the guide that suits your needs.

If you've created a guide, you can copy and paste the template below for me to add to the directory. When I post a dot, that means that all the guides have been added and a mod can hide the posts. If you are looking looking through the directory and you find a quick find code that is invalid, PLEASE let me know.

Thank you and enjoy!

***IMPORTANT NOTE: Some sections of the directory may be full! I try to keep it as up to date as possible but some sections (the misc section in particular) seems to stay full 99.9% of the time.
