Forum Help


This thread is for getting general help on the forums. This might be moving your thread, or dealing with some mild flaming/spamming. We aim to help players as soon as possible with all issues, but emergencies will be prioritised.

Here�s a list of the kind of things that can be reported in this thread:

✔ Spamming/Thread Derailing - sustained attempts to disrupt the forums.
✔ Persistent flaming � Without any seriously offensive language.
✔ Mild Offensive Language � Mild swearing, bypassing the filter
✔ Thread Content Copying

In addition you can use this thread to request any minor changes to your thread. This can include, but is not limited to:

✔ Thread Title Changes
✔ Moving Threads
✔ Locking a thread
