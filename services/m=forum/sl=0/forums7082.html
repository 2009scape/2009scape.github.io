Greetings!

This thread has some answers to following Seers medium task:

Kill one guard on each tower of the Ranging Guild using a longbow

Main things necessary for this task:

Range level: 40
REGULAR longbow (not from special woods like oak, maple, yew or magic)
About 300-400 iron arrows
Some prayer potions of food can be helpful, because killing with regular longbow takes some time

When enter center tower, you can`t leave, because if you leave kills are cleaned, and you need to start again.
Also necessary what You don`t have follower at time You doing this task (thanks for tip goes to Dog Man299).
Dsutc3 advice which some might consider noting before the task drink a sip of ranging potion, it should help hitting better and more often
Guards have timer, so if You not be fast killing them they disappears. If You exit tower, counter will reset, so if possible DO NOT EXIT TOWER.
You can call any friend to enter tower, to reset timer, so you can continue with guard killing without losing kill count.
Another solution from Pathlab is on 17 post: You can try to switch world without exiting tower to look world with rangers on towers.
Also Samzpoke and Hitaro9 (35 - 37 posts) suggest to use any other bow, and only to finish with long bow - that safes time and expences.
Zpm346 - suggest also to take range pots for low levels - thats of course helps


For those who find it useful- bump it - its always someone looks for help when this thread is some pages down
