Hello all,

Summary: I don't like Player-moderators being able to bypass my filter, when I've got it on 'off/friends' as it can disrupt my game-play at times. Solution add a toggle-feature that allows Moderators to toggle their ability on/off when it is necessary.

Welcome to my thread.. I'm ranting about something, which has been ranted about in the past.

Public Filter Bypassing:

Player-moderators have the "power/ability" to bypass my filter, even when set to "Friends" which highly annoys me.. As I have it set to friends for a reason(s).

Edit: I also found out that they may bypass my filter, even when I turn Public 'Off'.

Reason 1:
I don't want to be disturbed, unless it's a friend doing the disturbing.

Reason 2:
I wish to see only chat from my friends, not some random player. I also hate seeing half of a conversation.. When I have it set to friends, all I see is the moderators messages, which is kind of annoying to me.

I shouldn't need to switch Public to "on" just to be able to see the other half of the conversation, when I don't even want to see other player's conversations in the first place.

Reason 3:
Moderators spam, especially on Worlds 1, and 2. I shouldn't be forced to see their spam, when I've got my Public filtered.

Reason 4:
If "normal" players aren't allowed to bypass the filter, then moderators shouldn't be allowed to either.. As we are all supposed to be considered "equal" players.

------------------------------------------------------------

Solutions:

1) Simply remove their ability to bypass our filters, when it's set to "Friends/Off".

2) Increase the ignore list limit size to unlimited.

3) Add a toggle-feature, that allows moderators to toggle it on in extreme circumstances.

~Shia.

Please note.. I was tired when I wrote this, so forgive me if there are grammar or spelling errors.
