Ok so the rewards of Dungeoneering are very epic, but as a long time Slayer, I find myself spending a lot of time also picking up charms as well as bones.

Now considering the new Dungeoneering skill has the "bonecrusher" reward which automatically buries bones you receive from monster drops, I figure, why not make a similar reward for Summoning as well?

So my suggestion is this, the Charmchompa. An item which, when kept in your inventory, automatically picks up charms that would normally drop from a monster when it is slain. These charms would be placed in your inventory as if you picked them up yourself. Don't like a certain charm being collected by the Charmchompa? Simply right click on it and Toggle which charms you want collected.

Jagex could decide on its Dungeoneering level requirement, as well as how many tokens it would require.


Options
______


1. Toggle Charmchompa
2. Use Charmchompa
3. Drop Charmchompa
4. Examine Charmchompa


1. Toggle Charmchompa - an interface would pop up with a full list of charms you would like the Charmchompa to automatically pick up for you. You could toggle these on and off so you receive only the charms you want.

2. Use Charmchompa - no purpose apart from the fact almost all items have this option XD

3. Drop Charmchompa - self explanatory

4. Examine Charmchompa - Come on Chewy, Come on little Chewy. Ow! Bad Chewy! Bad Chewy!


I truly believe this would make such a great reward for Dungeoneering and would compliment the Bonecrusher reward very nicely. I'm sure it would become a favourite amongst all Slayers and Summoners alike.
