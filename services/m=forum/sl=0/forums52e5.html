Hey Everyone,

The following is a list of problems that have been experienced repeatedly by many players. If you see a problem that is frequently showing up please post it here and I will look into it and very likely add it to this list.

Please do not ask for direct help here, start up your own thread. It's much easier to help people with individual threads rathter than trying to answer 10 problems at once here.

***Technical: 404 Error on Client Download***

-->Details<--

There is a problem with one of the client download servers that is causing it to be impossible for certain computers to download the client from the Downloads and Wallpapers page. This appears to be a Jagex side problem unfortunately, all we can do is wait for a fix.

-->Update<--

Mod Drew has informed us that Jagex is aware of the problem and are working on a solution for it.

***Technical: Loaded Core Fonts***

-->Details<--

Many players have reported that their computers lock up at "loaded core fonts - 50%" and have been unable to load the game further. Any further information people can post is much appreciated.

-->Fixes<--

Some people have found that downloading the latest version of DirectX from the Microsoft website fixes this problem.

***Technical: MAC OS 10.6.4 Update**

-->Details<--

The MAC OS 10.6.4 update has been causing problems with the game. It is STRONGLY RECOMMENDED that you do not install this update until Jagex can release a workaround for the problems with it or Apple releases an update to correct this. This is a fault in the operating system with Nvidia graphics cards. Under any stress the system will crash. It is possible to play in safe mode even with this issue.

-->Sticky<--

See the sticky at 25-26-31-61212933 for more information.

***Technical: Unable to Pick Up Big Bones***

-->Description<--

There have recently been a very large number of reports of players when killing hill giants (mainly in the edgeville dungeon) they become unable to pick up the bones anymore. This is not a bug and is part of a system put in place by Jagex. Mod Mosley Explains things below.

-->Mod Mosley<--

This is not a bug. It is an intended part of gameplay, designed specifically to prevent the "farming" and regular transfer of some low cost items. This is all the details on this I can give you for the moment.

If you run into this message again, I advise doing a different activity for a while.

***Server: Worlds 29, 100, 170, and 171***

The servers for worlds 29, 100, 170, and 171 are offline, it's not a problem on the client side. Jagex is doing their best to get them back online as soon as possible. For now you will need to use other worlds until those ones can be restored.

***Technical: Cannot Load Runescape Website***

-->Description<---

Certain computers lose the ability to access only runescape.(c)om, jagex.(c)om and funorb.(c)om. All other websites function properly and other computers on the same network have no problems.

-->Sometimes Effective Fixes<--

Run Virus Scans

Edit the file below so that there is not mention of Runescape, Funorb, or Jagex in it:

C:\WINDOWS\system32\drivers\etc\hosts

Thank you very much to Pinguicula for finding this!

***Technical: Connection Lost on Returning to Lobby***

-->Description<--

For certain players when returning to the lobby they receive the "Connection Lost - Attempting to Reestablish" message and then are returned into the game making them unable to logout.

-->Fix<--

In most cases clearing the Jagex cache resolves this particular problem.

1) Close any open runescape or funorb windows.
2) Go to Start > My Computer
3) Open Local Hard Disk ( C: )
4) Say view files to any warning message
5) Look for a folder that says ".jagex_cache_32"
6) Delete that folder
7) Open a new browser window and load runescape normally.

If this does not resolve your problem, Jagex is investigating this and until it is fixed you will need to use the exit to logout button.

***Technical: Security Error, Code Will be Treated as Unsigned***

-->Description<--

For some users when loading the game they will receive a warning message saying that the code is unsigned or that they cannot create the cache folder. When this occurs users are stuck in safe mode. This seems to be an issue with the java security certificate or in some cases permissions on the cache folder.

-->Sometimes Effective Fixes<--

Create a new user account and load Runescape on it.
Replace the Java security certificate with one from a different computer.

-->Ineffective Fixes<--

Reinstall Java
Update Java
Rollback Java
Clear Caches
Update Graphics Card Drivers
Use Jagex Client
