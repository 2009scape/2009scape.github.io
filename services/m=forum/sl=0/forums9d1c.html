Updating Java

NOTE: There are a few alternative ways of updating your version of Java. We are providing Jagex's suggested way of updating Java.

Windows XP:

1. Go to Start > Control panel > Add/Remove programs
2. Uninstall all versions of Java
3. Remove any item with the Java Runtime Environment (JRE or J2SE) in the name.
4. Visit the Java website at http://www.java.com
5. Click �Free Java Download� button
6. If you are running the most up to date version, the website will specify!
7. If not, download the latest version
8. Install the update
9. Once installed, restart your computer

Vista/Windows 7:

Clearing Java Certificates

Windows XP

1. Go to Start > Control Panel > "Java"
2. Select 'Security' Tab
3. Click 'Certificates...'
4. From the menu that appears you can now select which certificates to delete.
5. Select 'Jagex Ltd'
6. Click Remove

Vista/Windows 7:

1. Go to Start > Control Panel > "Java"
2. Select 'Security' Tab
3. Click 'Certificates...'
4. From the menu that appears you can now select which certificates to delete.
5. Select 'Jagex Ltd'
6. Click Remove

Updating Graphics drivers

NOTE: There are a few alternative ways of updating your graphics card drivers. We our providing Jagex's suggested way of updating.

We strongly advise you only use your manufacturers authorised drivers rather than 3rd party drivers.

Windows XP:

To find out your graphics card manufacturer:
1. Click Start
2. Click Run
3. Type "DxDiag" (without the quotations) and hit enter
4. Click the display tab
5. Your graphics card name and model is next to name.

To Update:

1. Visit the website of your graphics card manufacturer
2. Visit their download section
3. Search for your graphics card model & name
4. Download graphics drivers
5. Install the new drivers
6. Restart computer
