This is our clans ranks from highest to lowest. you can gain ranks by:

1.Attending clan events regulary.

4.Bump the thread.

DO NOT ask for ranks.

~~~~~~~~~~~~~~~
General (Gold star):
~~~~~~~~~~~~~~~
Can run ANY event
Can decline and accept Applications
Can decline and accept War applications
Can kick from clan chat
Can decline or accept Ally requests

~~~~~~~~~~~~~~~~
Colonel (Silver star):
~~~~~~~~~~~~~~~~
Can run Dangerous Player vs Monster events (ie KBD, GWD)
Can decline and accept Applications
Can kick from clan chat
Can decline or accept Ally requests

~~~~~~~~~~~~~~~~~~
Lieutenant (bronze star):
~~~~~~~~~~~~~~~~~~
Can run any safe Player vs Player event and below (ie Castle Wars, Stealing Creation)
Can decline and accept Applications
Can kick from clan chat

~~~~~~~~~~~~~~~~
Sergeant (3 stripes):
~~~~~~~~~~~~~~~~
Can run any safe Player vs Monster event and below (ie. PC)
Can decline and accept Applications

~~~~~~~~~~~~~~~~
Corporal (2 stripes):
~~~~~~~~~~~~~~~~
Can run safe skill events (ie. Fishing Trawler, Blast Furnace)
Can decline and accept Applications

~~~~~~~~~~~~~~
No special Privileges
