(�.�� (�.Introduction

Welcome to the Gilded Altar Directory thread ! Before going any further, I would like to thank you all for using this thread to its fullest potential! Along with that, I would also like to thank Mod Howes and Mod Calm for their support toward the Directory. As without them, this thread would have never existed!

Anyways, the main functions of this thread are mainly to allow Gilded Altar hosters to post instant live alerts and to allow their POH to be on the Gilded Altar Directory. All of these might sound very overwhelming right now, but I promise that if we work together, everything will be very easy!

To hosts and goers...
Most of you are probably confused about the concept of Live Alerts right now. However, it is a very simply system that will allow you to achieve the best for your Gilded Altar Service. A Live Alert is mainly an alert that is created to alert everyone that something is currently occurring (in this case is the Gilded Altar service). Basically, you will be using this thread to post up all of your Gilded Altar services as Live Alerts !

To goers...
In the scenario that there are no Live Alerts posted, you may always refer to the thread Directory. The Directory will have information of hosts that will be available on certain day(s) of the week and certain time of the day to aid you. Remember that the hosts are only doing this voluntarily, so please do not be stressed if they are not on on the date they have promised.

BIG NOTE: If you have posted a Directory Template or asked questions on this thread, please PM me in-game to remind me if you see no response in awhile, thanks !

Please continue to the next page for the Live Alert and Directory Request Template !

(�.�� (�.Thread & Clan Chat Guidelines

� Thread Guidelines �
������������������

1. By posting your Live Alerts on this thread, you have agreed to allow us to add your name to the "Helpful Individuals" Listing. If you would like to not be added, please state so in your Live Alert.

2. Please only reply if you are posting a Live Alert or a Directory Request.

3. Please follow all three Honour, Respect, and Security rules. This includes the Forum Rules.

4. Please only post a Live Alert or Directory Request for yourself only. Do not post for your friends or for your other accounts!

5. Please refrain from posting multiple Live Alerts at the same time.

6. Please no fake Live Alerts!

7. Please only post a Live Alert if you have a Gilded Altar in your POH (nothing lower please!).

� Host Etiquette �
��������������
� Please use the right Live Alert format to not confuse your guests.

� Please try to have your private chat "on", so your guests may contact you if they have any question.

� Please do not charge your guests for using your POH features. Sharing is caring !

� Please keep the Burners litted if you have promised that when advertised.


� Guest Etiquette �
���������������
� Please be considerate to the host when you are in their house.

� Please ask if the Host needs anything such as Marentill herbs for the burners.

� Please always thank your host when you are done using their Glided Altar. There is nothing better than a sweet thank you message .

� Please volunteer to be a burner once awhile as the host might get tired after awhile.

� Please do not scold/flame your Host or Burner that they are inefficient at litting up the burners. This is very inappropriate as they are doing this voluntarily, so please avoid this.
