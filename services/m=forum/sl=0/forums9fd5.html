Mac - Guide


The Macintosh Operating System is one of the most user-friendly pieces of software commercially available.

Troubleshooting

Most issues with RuneScape can be solved by following this guide:

Note: Please only perform these steps if you are technically competent or a responsible adult. Jagex takes no responsibility for any data lost as a consequence of using this fix.

The operating system for Apple Macs (OS X) includes its own version of JAVA VM. This means that to update the VM that RuneScape uses, you will need to update the operating system.

To do this, use the Software Update or visit www.apple.com/support. From here, you will be able to update OS X.

The current update of OS X includes all solutions for sound and graphics problems that occur with JAVA.

Make sure your internet security programmes/firewall/router are configured to allow RuneScape to use your internet connection. Also, please make sure the following ports are unrestricted - 43594 and 43595 (for TCP traffic, both Inbound and Outbound). You will need to consult your user manual/help topics for the specific settings needed by your security programs/firewall/router.

If you are having problems with the game, please change your browser. Apple recommends the Safari browser, available at www.apple.com/safari

If you are having issues with RuneScape HD, make sure you have the latest drivers for your graphics card. These can be updated by using the Software Update.

Common Problems

Here are a few more specific issues that you may encounter when using a Mac to play RuneScape.

1. When loading a page on the RuneScape website, you may encounter a totally white screen:

Go back and make sure you have the latest Java installed. If you haven't, run Software Update. If the screen is still white when you load it, simply refreshing your browser should fix the problem.

2. When loading the RuneScape game, you may get a white box with a little red X on a document symbol in the top left of the box. This is a failed class loader and can be fixed by closing Safari and reopening it with the default settings.

3. Whilst playing the game, you may suddenly notice that your character and everything else stops moving and you can't click anything. Either run the Java GB or Garbage collector in the Java Console, or allocate more memory (RAM) to Java.

To allocate more memory, go to Applications ► Utilities ► Java Preferences. On the general tab, highlight the Java version at the top of the top box and hit the Options button ► enter/add: -Xmx200m

This will allocate 200mb of memory for Java so that it runs faster and this problem won't occur.

Do not put in a number that is greater than 50% of the RAM installed in your computer. For example, if you have 1024MB (or 1GB) of RAM installed, then the maximum you should enter is �-Xmx512m�.

4. Occasionally, you may be experiencing lag even if you have allocated more RAM to Java. If you are running iTunes or Microsoft Word, or both at the same time, you may have to close one since they tend to use a large amount of memory. If push comes to shove, buy more RAM.

5. Safari may randomly quit on you if you have several RuneScape windows open. What are you doing having more than one game window open? You will need to remember your RAM is limited, and if Java goes over the RAM cap you have set, Safari will close down to prevent your computer from reaching a RAM imbalance. Annoying as this is, it prevents undesirable effects of RAM-overload.

Browsers


Playing RuneScape on a Mac is very straightforward, it runs fine in both Safari (recommended by Apple) and Firefox.

Here's a few tips to get the best performance:

Safari:

Do not use tabs, use different windows. Although we do not recommend you do so, it should be possible to have the forums, the game, and several other windows open at the same time with no problems.

Remember, do not log in more than one account at any time as this is against the rules ("Multiple loggin-in".)


Firefox:

Firefox is very good and it's fast too. Even with multiple tabs open, it should run flawlessly with no lag (no spinning coloured wheel).

Using Your Mouse

To navigate RuneScape using your one button mouse, you have several options:


One button:

Setting the interface (the spanner tool next to Clan Chat) to one button will make everything in-game function as a secondary (or right) click. Not always the best idea, but can come in handy for some things.


Two buttons:

Setting the spanner to two buttons will mean that clicking is like a right click. To get the "info-click" (or right click) in RuneScape, hold the Apple key and then click.


Mighty Mouse:

The Mighty Mouse can be set to behave like a "PC mouse". Under System Preferences, select Mouse and then click Buttons. Primary click is like a left click, secondary is right.

This can be a bit hard to get used to, but eventually your Mighty Mouse will be a great tool you can use with ease.

OS X is also compatible with normal USB two button mice which will operate exactly like it would on Microsoft Windows.

General Maintenance

It's imperative that you run Software Update as often as possible. We recommend weekly runs to make sure everything on your Macintosh is up to date, this includes Java.

If you are ever having Java issues with your Mac and haven't run Software Update, run it! It can be found under System Preferences.

The newest version of Java should be downloaded if you don't already have it.


Current version of Java: J2SE 6.0

If you are having problems, switch back to 1.4.2 but continue using a signed applet.

Frequently Asked Questions

"What browser should I use to play on my Mac?"

The recommended browsers are Firefox, Safari and Shiira, though it should run fine in any browser.

Shiira generally has less bugs than Safari and has a footprint roughly half the size of Firefox. Safari is the browser recommended by Apple for general use.


"Can I download stuff while playing RuneScape on my Mac, or will it make me lag?"

Yes, however we cannot vouch for P2P programs causing/not causing RuneScape to lag. We do not recommend that you use them, as a lot of activity on such networks is illegal.

Don't download illegally. Ever. Downloading anything illegally will increase the risk of viruses on your Mac. They do exist. If you don't play it safe, you're going to be sorry.


"Don�t I have to download Java?"

No, Java is pre-installed on all Macs. If not, Software Update will pick it up for you.


"Will I have to configure anything to make RuneScape music work on my Mac?"

No, the music and sound effects should work with no problems.


"Can my Mac get keylogged?"

Technically yes, it is possible. However, as Macs should automatically detect any file that even has the remote possibility of being dangerous, your chances of getting a keylogger or a virus on a Mac are extremely slim.


"What about Spyware?"

See previous answer.


"How do I take a screenshot?"

Press Command-Shift-4, this will make your cursor turn into a crosshairs. Highlight the area you want to take a picture of and release the mouse button. The picture will automatically pop up on your desktop as a PNG image.

Command-Shift-3 takes a photo of your whole screen.
